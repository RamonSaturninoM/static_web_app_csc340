About This Site

This static site was built as a simple prototype using only HTML, CSS, and JavaScript, styled with Bootstrap. It demonstrates a searchable gallery, a details page, and a form that stores user‑added animals in the browser’s local storage.

You can use this as a starting point and customize the theme, typography, images, and content. Try adding pagination, categories, or a favorites list as stretch goals!
